<h1>On-premises Active Directory (Azure)</h1>
This tutorial outlines the implementation of on-premises Active Directory within Azure Virtual Machines to set certian users permissions to certian files.<br />

<h2>Environments and Technologies Used</h2>

- Microsoft Azure (Virtual Machines/Compute)
- Remote Desktop
- Active Directory Domain Services
- PowerShell

<h2>Operating Systems Used </h2>

- Windows Server 2022
- Windows 10 (21H2)

<h2>High-Level Deployment and Configuration Steps</h2>

- Select File and Users
- Set Permissions and Send
